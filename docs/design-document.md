# 

Director of Football is a simulation game, designed to put the user in the role of Director of Football in the football industry.
The game 

UI Menu Shortcuts

The bottom 10% of the screen will be half Advance button, half Menu. The menu will look similar to a mobile phone Home Screen, but cover only half of the screen. 

Inbox
- The core of the game. Below are examples of messages that the user will receive:

Unemployed 
- Messages from agent about opportunities
- Interview invitations 

Joining Team
- First Team List
- Academy List
- Depth Charts
- Fixtures
- Welcome Message from Chairman

Personnel
- Chairman
- (Contract Signing) Expectations for the contract period are defined, Performance targets are set 
- 
- Performance Updates
- (Quarterly) The Chairman will assess recent metric changes including wage budget, squad value changes, transfer negotiation success, squad harmony. 
- Manager
- Match Performance
- (After Each Match) The Manager will give an assessment of the performance and highlight any positive and negatives. 
- Squad Strength
- (Monthly) General discussions about the squad as a whole. Manager has to address performance and DOF can highlight any concerns.
- Requirements
- (Before Transfer Windows) The Manager will give Requirements about the squad changes that they would like. The Manager's Club Influence has an effect on the importance of the Requirement, and there is also severity and priority of each Requirement. 
- Assistant Manager
- Squad Harmony
- 
- Coaches
- Training Performances
- Chief Scout
- Transfer Targets
- (Weekly) Recruitment Meeting where focuses are set, targets are assigned and updated on, agent meetings are debriefed. 
- Head of Development
- (Weekly) A weekly summary of Youth Team developments, recommendations of promotions and players to cut, and matches to attend. Interesting players from other teams can also be brought up to add as targets.  

Market
- League/World Transfer Updates
- Shortlist Updates
- Agent Correspondence Summary

Player Profile
- Overview: distilled and prioritized information from the other panels
- Scout Report: Player's style and traits
- Analyst Report: Player's seasonal and career metrics
- Coach Report: Player's Personality, attitude, tactical fit
- Medical Report: Player's fitness, condition and injuries
- Player Liaison Officer: Player's relationships, and happiness
- Contract: Contract Details, Promises
- Transfer: Transfer Value, Value History, Interested Clubs
- History
