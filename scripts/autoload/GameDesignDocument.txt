1. Gameplay Loop:

a) Day-to-Day Loop:
* Morning:
   * Wake up to a dynamic news feed on a ticker at the bottomo of the screen
   * Review prioritized emails and messages on the in-game phone
   * Attend to urgent matters (injuries, transfer requests, etc.) via quick-response options
   * Review interactive training reports with 3D heatmaps and adjust schedules using an intuitive drag-and-drop interface
* Midday:
   * Engage in dynamic staff meetings with dialogue options influencing staff morale and performance
   * Make decisions on transfer targets using an advanced scouting network with detailed player analytics
   * Review and approve youth academy activities through a mini-game style interface
* Afternoon:
   * Handle media interactions via a press conference simulator with reputation impact
   * Work on long-term projects using a strategy game-like interface for facility improvements and scouting network expansion
   * Analyze performance data and statistics using interactive charts and AI-driven insights
* Evening:
   * Attend matches with a 'Director's Box' view, offering unique perspectives and decision-making opportunities
   * Review personalised end-of-day reports with actionable insights
   * Plan for the next day using an AI assistant that suggests optimal schedules based on club needs

b) Week-to-Week Loop:
* Pre-match:
   * Engage in tactical discussions with the manager using a 3D pitch simulator
   * Review opponent scouting reports with video highlights and tactical breakdowns
   * Handle time-sensitive transfer or contract negotiations with a ticking clock element
* Match Day:
   * Experience matches through multiple viewing options (Director's Box, Sideline, Tactical Overview)
   * Provide halftime input to the manager through a dialogue system with tactical implications
   * Conduct post-match analysis using interactive heat maps and player performance metrics
* Post-match:
   * Review comprehensive match reports with video highlights and AI-generated insights
   * Address match-related issues through a crisis management mini-game
   * Plan for the upcoming week using a strategic calendar interface

c) Season-to-Season Loop:
* Pre-season:
   * Set season objectives through negotiations with the board, influencing budget and expectations
   * Conduct squad building through an engaging transfer market simulation
   * Arrange pre-season activities via an interactive global map
* Mid-season:
   * Navigate the intense January transfer window with real-time market fluctuations
   * Participate in a mid-season board meeting with potential storyline implications
   * Oversee youth intake through a scouting mini-game and development strategy planner
* End of season:
   * Experience an immersive season review with the board, potentially affecting your job security
   * Manage contract renewals and releases through a detailed negotiation system
   * Plan for the next season using predictive analytics and scenario planning tools

* Home Screen:

* Apps:
   * Messages: Threading system with read receipts and emoji reactions
   * Email: Priority inbox with AI-sorted categories
   * Calendar: Interactive timeline with drag-and-drop event management

* Customizable modular design with drag-and-drop widgets

* AI-driven insights and suggestions based on club performance

c) Match View:
